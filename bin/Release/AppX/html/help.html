Sudoku help


how to play

Sudoku는 가로·세로 9칸씩 총 81칸으로 이뤄진 정사각형의 모든 가로·세로줄에, 그리고 가로·세로 3칸씩 총 9칸으로 이뤄진 9개의 작은 사각형 안에 1~9의 숫자를 겹치지 않게 적어 넣는 숫자 퍼즐입니다.
1. PC에서는 오른쪽 마우스 태블릿에서는 기계 하단부분에서 화면위로 제스쳐를 해줍니다. 그러면 게임에 필요한 메뉴가 뜹니다.
2. NEW GAME 버튼은 새로운 숫자를 나열해 새로운 게임을 하는 것입니다.
3. RESET 버튼은 새로운 숫자를 나열하지 않고 사용자가 입력한 부분에 글자를 지워주고 다시 하실수 있습니다. 따라서 시계도 초기화 됩니다.
4. SAVE 버튼은 게임중인 sudoku 게임을 저장하여 나중에 이어 하실수 있습니다.
5. LOAD 버튼은 마지막으로 저장했던 게임을 불러와 게임을 이어 하실수 있습니다.
6. All Note 버튼을 누르게 되면 좀더 Sudoku를 쉽게 즐길 수 있도록 해당 칸에 입력가능 모든 수를 보여줍니다.
7. Clear Note 버튼은 ALL Note 기능으로 표시된 숫자들을 전부 삭제해주는 기능입니다.
8. 시계 버튼을 누르면 게임을 잠시 멈출수 잇습니다.C
